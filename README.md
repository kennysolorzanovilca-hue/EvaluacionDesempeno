# Evaluación de Desempeño

Esta es una aplicación web para realizar evaluaciones de desempeño anual.

## Estructura del Proyecto

El proyecto está estructurado de la siguiente manera:

- `index.html`: El archivo principal de la aplicación.
- `css/style.css`: Contiene los estilos de la aplicación.
- `js/`: Contiene los archivos de JavaScript.
  - `main.js`: El punto de entrada de la aplicación.
  - `competencies.js`: Contiene los datos de las competencias.
  - `ui.js`: Maneja la renderización de la interfaz de usuario y la manipulación del DOM.
  - `state.js`: Maneja el estado de la aplicación.
  - `utils.js`: Contiene funciones de utilidad.
  - `echarts-config.js`: Configura y renderiza el dashboard de ECharts.

## Cómo ejecutar la aplicación

Simplemente abra el archivo `index.html` en su navegador web.
"# EvaluacionDesempeno"
